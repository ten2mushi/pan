# Research Notes Index — Real-Time Embedded Audio Signal Processing

**Philosophy.** Every note prioritizes **minimizing bytes moved through the memory hierarchy** while delivering deterministic real-time behavior on embedded targets (Cortex-M/A, RISC-V, DSPs). Notes are written from first principles with full mathematical derivations, cache-traffic accounting, SIMD mappings, and verified citations.

## Citation & Cross-Reference Conventions

- Primary papers are cited with full author list, year, title, venue, DOI (verified).
- **[derived]** = arithmetic from formulas in the note.
- **[speculative]** = engineering extrapolation, not published measurement.
- Internal links use relative paths from the note’s directory (e.g. `../transforms/discrete-fourier-transform.md`).
- When a note corrects earlier claims, a prominent “Provenance / Corrections” box appears after the abstract.
